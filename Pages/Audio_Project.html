Audio Project

This project was created for my module CMP407 at university. This module looks at audio tools in the gaming sector along with audio programming in C++.

For this project we were allowed to use any engine we wanted so I chose Unity. My game is a simple game that demonstrates a number of audio techniques such as occlusion and spacialisation. My game is set in a prision with different sounds in different areas.
