AR Project

This project was created for my module CMP404 at university. This module looks at emerging technologies in the gaming sector and specifically AR.

For this project we used Unreal Engine 4. My game is a simple game where you defend a base, in this case the Android mascot, for waves of enemies. You can swap weapons by using the UI buttons or looking at the provided images. You can shoot down planes that rotate around the mascot to get health back.
