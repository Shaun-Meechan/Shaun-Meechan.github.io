Gameplay Mechanics Development

This project was created for my module CMP302 at university. This module focuses on the principles of formalising, implementing, testing and iterating on functional game mechanics applied in the context of a game engine.

For this project we used Unreal Engine 4. My mechanic was a system that belended RTS gameplay with FPS gameplay. The player creates units then can take direct control of them. This is similar to the mechanic used in Battlefield 2: Modern combat.

This small proof of concept shows the ability to spawn a generic soldier and use their weapon and abilities.
