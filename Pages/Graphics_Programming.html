Graphics Programming

This is a project that was for my graphics programming module, CMP301 in my 3rd year. It features:

  • Vertex manipulation - heightmap.
  • Post processing - bloom and chromatic abberation.
  • Lighting and shadows - directional light, point light, spotlight.
  • Tessellation - tesselated heightmap
  • Use of the geometry shader - billboard trees.
